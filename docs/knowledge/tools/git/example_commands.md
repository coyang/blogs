# Example Commands

## Change commit log

- Change commit time

```bash
git commit --amend --date="Fri Oct 13 12:01:13 2023 +0300" --no-edit
git commit --amend --date=now --no-edit
```

- Change commit author

```bash
git commit --amend --author="Author Name <author@email.com>" --no-edit
```
